1. Adjoint Method를 활용한 Optimal Control Problem 풀이

1.1. Solving Optimal Control Problem

다음과 같은 Contrained Optimal Control Problem을 Lagrage Multiplier를 통해 푸는 방법을 소개합니다 1.

\[\min_u \int_{t_0}^{t_1} f(t, x(t), u(t))~dt\]

subject to

\[\begin{split}\begin{cases} x'(t) &= g(t,x(t), u(t))\\ x(t_0)&=x_0 \end{cases}\end{split}\]

가장 간단한 방법론은 다음과 같습니다.

  1. 해당 Optimal Control Problem의 Hamiltonian을 구성한다.

    \[H = f(t, x(t), u(t)) + \lambda(t)\cdot g(t, x(t), u(t))\]

  2. Optimality condition

    \[\frac{\partial H}{\partial u} = 0 \text{ at } u^* \Rightarrow f_u + \lambda g_u = 0\]

  3. Adjoint equation

    \[\lambda' = -\frac{\partial H}{\partial x} \Rightarrow \lambda' = -(f_x + \lambda g_x)\]

  4. Transversality condition

    \[\lambda(t_1) = 0\]

  5. State equation

    \[x' = g(t,x,u) = \frac{\partial H}{\partial \lambda}, ~x(t_0)=x_0\]

참고로 Pontryagin’s maximum principle은 다음과 같습니다.

참고

If \(u^*(t)\) and \(x^*(t)\) are optimal for the above optimal control problem, then there exists a piecewise differentiable adjoint variable \(\lambda(t)\) such that

\[H(t, x^*(t), u(t), \lambda(t)) \leq H(t, x^*(t), u^*(t), \lambda(t))\]

for all controls \(u\) at each time \(t\), where the Hamiltonian \(H\) is

\[H = f(t, x(t), u(t)) + \lambda(t)\cdot g(t, x(t), u(t))\]

and

\[\begin{split}\lambda'(t) & = - \frac{\partial H(t, x^*(t), u^*(t), \lambda(t))}{\partial x}\\ \lambda(t_1) &= 0\end{split}\]

1.2. Example 1

\[\min_u \int_0^1 u^2(t)dt\]

subject to

\[\begin{split}x'(t) &= x(t) + u(t)\\ x(0) &= 1\end{split}\]

참고

  1. 해당 Optimal Control Problem의 Hamiltonian을 구성한다.

    \[\begin{split}H &= f(t, x(t), u(t)) + \lambda(t)\cdot g(t, x(t), u(t))\\ &= u^2 + \lambda(x+u)\end{split}\]

  2. Optimality condition

    \[\begin{split}\frac{\partial H}{\partial u} &= 0 \text{ at } u^* \Rightarrow f_u + \lambda g_u = 0\\ 2u + \lambda &=0 \Rightarrow u^* = -\frac{1}{2} \lambda\end{split}\]

  3. Adjoint equation

    \[\begin{split}\lambda' &= -\frac{\partial H}{\partial x} \Rightarrow \lambda' = -(f_x + \lambda g_x)\\ &=-\lambda \Rightarrow \lambda(t) = c \cdot e^{-t}\end{split}\]

  4. Transversality condition

    \[\begin{split}\lambda(t_1) &= 0\\ \lambda(1) &= c\cdot e^{-1} = 0 \\ &\Rightarrow c=0 \\ &\Rightarrow \lambda=0 \\ &\Rightarrow u^*=-\frac{1}{2}\lambda = 0\end{split}\]

  5. State equation

    \[\begin{split}{x^{*}}' &= g(t,x^*,u) = \frac{\partial H}{\partial \lambda}, ~x^*(t_0)=x_0\\ {x^{*}}' &= g(t,x^*,u) = \frac{\partial H}{\partial \lambda}, ~x^*(0)=1\\ x^* &= e^t\end{split}\]

1.3. Example 2: Forward-Backward Sweep Method

첫번째 예제와 같이 \(H_u=0 \text{ at } u^*\) 를 통해서 3개의 Unknowns를 2개로 소거하는 경우는 많지 않습니다. 그런 경우에는 Forward-Backward Sweep 1 이라는 Iterative Scheme을 사용합니다.

  1. \(u^*\) 의 초기추정값(\(u_0\))을 설정합니다.

  2. 초기추정값(\(u_0\))을 이용하여 \(x_0\) 의 방정식인 State differential equation을 풉니다. (Forward)

  3. 앞에서 구한 \(u_0\) 와 \(u_0\) 를 활용하여 \(\lambda_0\) 의 방정식인 Adjoint differential equation을 풉니다. (Backward)

  4. \(H_u(u_{new}) = 0\) 을 풀어서 얻은 \(u_{new}\) 과 \(u_0\) 을 적절한 방식으로 \(u_1\) 을 업데이트 합니다.

    \[u_1 = \frac{1}{2}\left(u_{new} + u_0\right)\]

  5. Convergence를 체크하고, 수렴할때까지 위의 단계들을 반복합니다.
